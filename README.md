# inaturalist-image-scraper
A Python scraper for downloading species images from iNaturalist on macOS
## 🚀 如何在 Mac 上运行此脚本

本项目是一个适用于 macOS 的 iNaturalist 图像抓取工具。请按照以下步骤配置环境并运行：

### 1. 安装所需的依赖库
在运行脚本之前，需要先安装 Python 的基础数据处理和网络请求库。
请打开电脑的终端（Terminal 或 Cursor 里的 New Terminal），复制并运行以下命令：

```bash
python3 -m pip install pandas requests

2. 运行抓取脚本
确保你的终端路径已经切换到代码所在的文件夹，然后运行以下命令启动程序：
python3 download.py
