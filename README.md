**iNaturalist Image Downloader / iNaturalist 图像抓取工具**

一个适用于 macOS 的 iNaturalist 图像抓取工具。
A macOS tool to download images from iNaturalist.

📦 安装依赖 / Install Dependencies

在运行脚本之前，请先安装 Python 的基础库。
Before running the script, install the required Python libraries.

打开终端（Terminal 或 Cursor 中的 New Terminal），然后运行：

**python3 -m pip install pandas requests**

▶️ 运行脚本 / Run the Script

确保终端的当前路径切换到代码所在文件夹，然后运行：
Make sure your terminal is in the folder where the code is located, then run:

**python3 download.py**

程序将开始从 iNaturalist 下载图片。
The program will start downloading images from iNaturalist.

💡 注意事项 / Notes

需要安装 Python 3 及以上版本。
Python 3 or above is required.

网络连接必须正常，以便访问 iNaturalist API。
A stable internet connection is required to access the iNaturalist API.

下载的图片会保存在脚本所在的目录下。
Downloaded images will be saved in the same directory as the script.
